- Weather app

- Basic weather app that shows:
- Date, time, location, min/max temps in celsius, and fahrenheit
- Technologies used: HTML/SASS/JavaScript, and an API from openweathermap
- RUN PROJECT:
- a) write npm start in terminal
- b) use live server
